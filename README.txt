地理一問一答アプリ 300問版

大学受験地理の反復学習用Webアプリです。
ZIPを展開し、index.html をブラウザで開くと起動します。

収録問題数：300問
単元構成：
・地形 30問
・気候 30問
・農業 30問
・工業 30問
・資源・エネルギー 30問
・人口・都市 30問
・交通・通信・貿易 30問
・国家・民族・宗教 30問
・地誌 30問
・地図問題 30問

主な機能：
・単元別出題
・難度別出題
・ランダム出題
・今日の目標問題数
・解答表示
・解説表示
・ヒント表示
・自己判定（できた／あやしい／できない）
・苦手問題の自動抽出
・間違えた問題だけ復習
・あやしい問題だけ復習
・問題検索
・タグ別復習
・単元別正答率
・連続学習日数
・日別学習履歴
・学習データのエクスポート／インポート
・学習記録の保存
・スマホ向け画面設計
・PWA用manifestとservice worker

問題を追加・修正する場合は data フォルダ内の単元別 questions-*.js を編集してください。
各問題は id, categoryGroup, category, unit, type, level, question, choices, answer, explanation, hint, tags, reviewPoint を持ちます。
categoryGroup と category は、既存の unit から自動補完されます。
